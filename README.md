# Time Keeper v2

Time Keeper v2 is a simple, portable command-line time tracker written in Python.  
It lets you clock in/out, track multiple projects, switch users, and view basic reports.

---

## Features

- Clock in / clock out  
- Multi-user tracking  
- One JSON log file per project  
- Daily, weekly, and monthly summaries  
- Automatic folder creation  
- Fully portable (run from any folder or drive)

---

## How to use

1. Run the script:

```bash
python time_keeper_v2.py
```

2. Enter:
   - project name  
   - optional custom folder for logs  
   - username  

3. Use the menu to:
   - start sessions  
   - end sessions  
   - check status  
   - view summaries  
   - switch users  
   - quit

---

## Requirements

- Python 3.10+
- No external libraries needed
